# Coffee Shop

Our coffee shop offers an easy-to-use online platform for customers to reserve tables, order coffee and food, and track the status of their order in real-time. We provide a hassle-free experience that ensures the best service possible for our customers.

## Tech Stack

**Client:** Typescript, Javascript, React, Redux, TailwindCSS...

**Server:** PHP, laravel, SQLITE
